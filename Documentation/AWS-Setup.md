# AWS Infrastructure Setup

This document explains the AWS infrastructure created for a secure hybrid cloud environment connecting an on-premises office network to AWS using a Site-to-Site VPN.

## Services Used

- Amazon EC2
- Amazon VPC
- Internet Gateway
- Route Tables
- Security Groups
- Site-to-Site VPN
- Virtual Private Gateway
- Customer Gateway

## Purpose

Deploy a Linux-based proxy server in AWS and securely connect it to the office network using an IPSec VPN through a Sophos XGS Firewall.
